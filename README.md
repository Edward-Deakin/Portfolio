## Stack

- [Astro](https://astro.build) with React integration for a plug-and-play component system
- [shadcn/ui](https://ui.shadcn.com) components paired with TailwindCSS used for its clean pre-built components
- TypeScript, JS, HTML and CSS languages used
- Code deployed on [Zerops](https://zerops.io)

## Local Deployment

These commands will clone the public repository locally and then install all the dependencies required to run the project:

```bash
git clone https://github.com/<your-username>/portfolio.git
cd portfolio
npm install
npm run dev
```
